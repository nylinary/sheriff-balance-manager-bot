# Sheriff Balance Manager Bot

Telegram-бот для учёта денежных операций по нескольким валютам/счетам с разделением доступа.

## Возможности

- Внесение операций сотрудниками в групповом чате (`/руб 3200`, `/доллс -5000`)
- Баланс скрыт в рабочем чате — только короткое подтверждение
- Приватный интерфейс для админов: кошелёк, история, откат, Excel-выгрузка
- Временные окна доступа на внесение операций (`/открытьд 19:00-22:00`)
- Пагинация истории с inline-кнопками
- Защита от двойного отката

## Валюты

| Команда     | Валюта         | Emoji |
|-------------|----------------|-------|
| `/руб`      | Руб            | 🇷🇺    |
| `/доллс`    | Доллар синий   | 💵    |
| `/доллб`    | Доллар белый   | 💴    |
| `/евро`     | Евро           | 🇪🇺    |
| `/евро500`  | Евро500        | 💶    |

Конфигурация валют — в `bot/config/currencies.py`.

## Команды

### Рабочий чат (сотрудники и админы)

| Команда | Описание |
|---------|----------|
| `/<валюта> <сумма>` | Внести операцию (положительное/отрицательное целое число) |

### Личный чат (только админы)

| Команда | Описание |
|---------|----------|
| `/кошелек`, `/дай` | Текущий баланс по всем валютам |
| `/история` | История операций с пагинацией |
| `/история @username` | История по конкретному сотруднику |
| `/открытьд 19:00-22:00` | Открыть доступ на период |
| `/закрытьд` | Закрыть доступ досрочно |
| `/выгрузка`, `/excel` | Выгрузка всех операций в `.xlsx` |

## Запуск

### Требования

- Docker + Docker Compose
- Telegram Bot Token (через [@BotFather](https://t.me/BotFather))

### Шаги

```bash
cp .env.example .env
```

Заполнить `.env`:

```env
BOT_TOKEN=123456:ABC-DEF
ADMIN_IDS=123456789,987654321
TIMEZONE=Europe/Moscow
```

Запуск:

```bash
docker compose up --build
```

Бот автоматически создаёт таблицы при первом старте. Для production используйте миграции:

```bash
docker compose exec bot uv run alembic upgrade head
```

## Стек

- Python 3.12
- aiogram 3.x
- SQLAlchemy 2.x + asyncpg
- PostgreSQL 16
- Redis 7 (FSM storage)
- Alembic (миграции)
- openpyxl (Excel)
- uv (управление зависимостями)
- Docker Compose

## Структура проекта

```
bot/
  config/         — настройки, конфигурация валют
  models/         — SQLAlchemy модели (User, Balance, Operation, AccessWindow)
  repositories/   — слой доступа к данным
  services/       — бизнес-логика
  handlers/       — Telegram-обработчики команд и callback'ов
  keyboards/      — inline-клавиатуры
  utils/          — форматирование, парсеры, работа со временем
alembic/          — миграции БД
```
